# Taiwan - Accounting


This is the base module to manage the accounting chart for Taiwan in Odoo.
==============================================================================
    

## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_tw
```

## Dependencies

- account
- base_address_extended
