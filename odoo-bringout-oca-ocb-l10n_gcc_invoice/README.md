# Gulf Cooperation Council - Invoice


Adds Arabic as a secondary language on your invoice, credit note, debit note, vendor bill, and refund bill


## Installation

```bash
pip install odoo-bringout-oca-ocb-l10n_gcc_invoice
```

## Dependencies

- account
